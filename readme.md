# Labartory Task 4

Just a simple but convenient android day-counter widget with on-event-day notifications, developed for educational purposes.


### Cool Features
- The widget keeps track of time or time zone changes and updates the day-counter as required.
- The user should never worry about losing notifications or seeing an outdated day-counter, as the widget will automatically reschedule any notifications or day-counter updates after phone restart.
- The widget is battery power efficient and doesn’t cause unnecessary phone wakeups.
- Modern and stylish design that will fit in with most homescreen themes and wallpapers.
